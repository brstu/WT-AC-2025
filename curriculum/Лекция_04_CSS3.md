## Материал для лекции

### 1. Основы CSS, селекторы, наследование
**Теория:**  
- CSS управляет стилями элементов: цвет, размер, отступы, шрифты.
- Селекторы: по тегу (`div`), классу (`.class`), id (`#id`), атрибуту (`input[type="text"]`).
- Наследование: дочерние элементы наследуют некоторые свойства от родителя.
- Каскадность: приоритеты — inline > id > class > тег.
- **Пример:**  
  ```css
  body { font-family: Arial; }
  .main { color: blue; }
  #header { background: #eee; }
  ```

### 2. Flexbox и Grid
**Теория:**  
- Flexbox: удобен для одномерных (в строку или столбец) макетов.
  - Контейнер: `display: flex;`
  - Управление выравниванием, порядком, переносом.
  - **Пример:**  
    ```css
    .row { display: flex; }
    .item { flex: 1; }
    ```
- Grid: двумерная сетка, удобно для сложных макетов.
  - Контейнер: `display: grid;`
  - Задание строк и колонок, размещение элементов.
  - **Пример:**  
    ```css
    .grid { display: grid; grid-template-columns: 1fr 2fr; }
    ```

### 3. Препроцессоры, адаптивность
**Теория:**  
- Препроцессоры (Sass, Less): переменные, вложенность, миксины, функции.
- Адаптивная верстка: media queries для разных экранов.
- **Пример:**  
  ```css
  @media (max-width: 600px) {
    .menu { flex-direction: column; }
  }
  ```
- Использование переменных в Sass:
  ```scss
  $main-color: #3498db;
  body { color: $main-color; }
  ```
