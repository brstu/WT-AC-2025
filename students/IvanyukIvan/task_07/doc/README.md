## Ход выполнения работы

### 1. Структура проекта

- `doc/README.md` — отчёт по лабораторной работе
- `doc/screenshots/` — папка под скриншоты (используются ссылки на открытые изображения)
- `src/package.json` — зависимости и скрипты проекта
- `src/vite.config.js` — конфигурация Vite
- `src/index.html` — точка входа Vite
- `src/src/` — исходный код React-приложения (компоненты, страницы, стили, тест)

### 2. Реализованные элементы

- Маршрутизация React Router v6 (список, детали, создание, редактирование, 404)
- Управление данными через TanStack Query с простыми состояниями загрузки/ошибок
- Формы создания/редактирования с примитивной валидацией
- Простые фильтры списка по строке поиска
- Минимальные уведомления через alert
- Один базовый тест на vitest
