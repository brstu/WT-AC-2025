# Портфолио фотографа

## Описание

Профессиональное портфолио фотографа с интерактивными компонентами на чистом JavaScript.

![Внешний вид](preview.png)

## Цели

- Работа с DOM и событиями
- Делегирование событий
- Доступность (a11y)
- Валидация форм
- Интерактивная галерея

## Реализованные компоненты

### 1. Табы фотографических серий

- Переключение между сериями: "Портреты", "Природа", "Город"
- Динамическое создание галереи
- Клавиатурная навигация
- ARIA: `role="tablist"`, `aria-selected`, `aria-controls`

### 2. Аккордеон "Обо мне"

- Раскрывающиеся секции: "Биография", "Образование", "Достижения"
- Плавная анимация раскрытия
- ARIA: `aria-expanded`, `aria-controls`, `role="region"`

### 3. Модальное окно слайд-шоу

- Просмотр фотографий в полноэкранном режиме
- Навигация между изображениями
- Информация об EXIF данных
- Tooltip с техническими характеристиками снимков

### 4. Бургер-меню

- Адаптивное меню для мобильных устройств
- Плавное открытие/закрытие
- ARIA: `aria-expanded`, `aria-controls`

### 5. Форма обратной связи с валидацией

- Поля: имя (обязательное), email (валидный), сообщение (20+ символов)
- Валидация в реальном времени
- Кнопка отправки блокируется при ошибках
- Пользовательские сообщения об ошибках

## Проверка функциональности

**Табы серий:**
- Клик по вкладкам "Портреты", "Природа", "Город"
- Клавиши Tab для навигации между табами
- Enter/Space для активации

**Галерея:**
- Клик по изображениям для открытия слайд-шоу
- Стрелки ← → для навигации между фото
- Esc для закрытия модального окна
- Tooltip с EXIF данными при наведении

**Аккордеон:**
- Клик по заголовкам для раскрытия секций
- Enter/Space для управления с клавиатуры

**Бургер-меню:**
- Клик по иконке меню на мобильных устройствах
- Автоматическое закрытие при клике вне меню

**Форма:**
- Попробуйте отправить с пустыми полями
- Проверьте валидацию email
- Убедитесь, что сообщение требует минимум 20 символов
- Наблюдайте активацию/деактивацию кнопки отправки

## Доступность

**Клавиатурная навигация:**
- Tab/Shift+Tab - навигация по элементам
- Enter/Space - активация элементов
- Стрелки ← → - навигация в галерее и табах
- ESC - закрытие модального окна

**ARIA-атрибуты:**
- `role="tablist"`, `role="tabpanel"`, `role="tab"` - для табов
- `aria-selected`, `aria-controls` - состояние табов
- `aria-expanded` - для аккордеона и бургер-меню
- `aria-hidden` - для скрытых элементов
- `aria-modal="true"` - для модального окна
- `role="alert"` - для сообщений об ошибках

## Удобство интерфейса

- Адаптивный дизайн для всех устройств
- Визуальная обратная связь при взаимодействии
- Четкие состояния фокуса для навигации с клавиатуры
- Интуитивная навигация по галерее

**Делегирование событий:**
- Обработчик кликов на галерее повешен на контейнер

**Валидация формы:**
- Валидация при вводе и при отправке
- Динамическая блокировка кнопки отправки
- Пользовательские сообщения об ошибках

**Модальное окно:**
- Блокировка прокрутки страницы при открытии
- Фокус управления внутри модального окна
- Поддержка навигации с клавиатуры
