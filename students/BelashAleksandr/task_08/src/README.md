# Приют для питомцев

Веб-приложение для просмотра питомцев из приюта.

## Запуск

### Локально

```bash
python -m http.server 8000
```

### Docker

```bash
docker-compose up
```

### Тесты

```bash
npm install
npm test
```

## Структура

- `index.html` - главная страница
- `app.js` - основная логика
- `__tests__/` - тесты
- `Dockerfile` - конфигурация Docker

## Технологии

- HTML/CSS/JavaScript
- Jest
- Docker
- GitHub Actions
