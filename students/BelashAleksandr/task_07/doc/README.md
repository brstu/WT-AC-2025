
### 2. Реализованные элементы

**Основной функционал:**

- **Маршрутизация с React Router v6:**
  - Главная страница со списком питомцев (`/`)
  - Страница детальной информации о питомце (`/pet/:id`)
  - Страница добавления нового питомца (`/add`)
  - Страница редактирования питомца (`/edit/:id`)
  - Страница 404 для несуществующих маршрутов

- **CRUD операции:**
  - Просмотр списка питомцев
  - Просмотр детальной информации
  - Добавление нового питомца
  - Редактирование существующего питомца
  - Удаление питомца

- **Управление состоянием:**
  - Использование React useState для локального состояния
  - Хранение данных в localStorage
  - Управление состояниями загрузки (loading)

- **Формы:**
  - Форма добавления питомца с полями: кличка, тип, возраст, порода, описание, изображение
  - Форма редактирования с предзаполнением данных
  - Базовая валидация обязательных полей

- **Аутентификация:**
  - Простая система входа/выхода через prompt
  - Защищенные маршруты для добавления и редактирования (доступны только авторизованным)
  - Хранение информации о пользователе в localStorage

- **Дополнительные возможности:**
  - Фильтрация питомцев по типу и статусу (все, собаки, кошки, доступны, пристроены)
  - Изменение статуса "пристроен" для питомца
  - Изображения питомцев из открытых API (Dog CEO API, The Cat API)
  - Уведомления через alert при операциях

## Запуск проекта

1. Перейти в директорию `src`:

```bash
cd src
```

1. Установить зависимости:

```bash
npm install
```

1. Запустить проект в режиме разработки:

```bash
npm run dev
```

1. Открыть браузер по адресу: `http://localhost:5173`

## Сборка для продакшена

```bash
npm run build
```

## Вывод

В ходе выполнения лабораторной работы был создан полноценный SPA каталог питомцев из приюта на React. Реализована маршрутизация с использованием React Router v6, включая вложенные маршруты и защищенные маршруты для авторизованных пользователей.

Освоены навыки работы с формами, валидацией данных, управлением состоянием через React hooks (useState, useEffect). Реализован полный цикл CRUD операций с данными, хранящимися в localStorage.

Приложение имеет интуитивно понятный интерфейс с навигацией, фильтрацией данных и уведомлениями о выполненных операциях. Использованы открытые API для получения изображений животных, что делает каталог более наглядным и привлекательным.
