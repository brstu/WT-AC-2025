# Установка и запуск

## Шаги установки

1. Скопировать `.env.example` в `.env`
2. Установить зависимости: `npm install`
3. Создать миграции: `npm run migrate`
4. Заполнить БД: `npm run seed`
5. Запустить сервер: `npm start`
