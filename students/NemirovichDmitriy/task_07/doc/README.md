## Цель работы

Создать полноценный SPA на React с маршрутизацией, управлением состоянием, формами и взаимодействием с API.

## Ход выполнения работы

### 1. Структура проекта

Проект разделён на две основные директории:

- `doc/` — документация и скриншоты
  - `README.md` — данный отчёт
  - `screenshots/` — скриншоты приложения
- `src/` — исходный код приложения
  - `package.json` — зависимости проекта
  - `vite.config.js` — конфигурация Vite
  - `.env` — переменные окружения
  - `index.html` — главная HTML-страница
  - `main.jsx` — точка входа React
  - `App.jsx` — главный компонент приложения
  - `pages/` — компоненты страниц
    - `CompaniesList.jsx` — список компаний
    - `CompanyDetail.jsx` — детали компании
    - `NewCompany.jsx` — создание компании
    - `EditCompany.jsx` — редактирование компании
    - `VacanciesList.jsx` — список вакансий
    - `VacancyDetail.jsx` — детали вакансии
    - `NewVacancy.jsx` — создание вакансии
    - `NotFound.jsx` — страница 404

### 2. Реализованные элементы

**Функциональность:**

- Список компаний с карточками
- Просмотр детальной информации о компании
- Добавление новой компании через форму
- Редактирование существующей компании
- Список вакансий с фильтрацией по городу
- Просмотр детальной информации о вакансии
- Добавление новой вакансии
- Отклики на вакансии через формы
- Страница 404 для несуществующих маршрутов

**Технологии:**

- React 18
- React Router v6 для маршрутизации
- Vite в качестве сборщика
- Inline-стили для оформления
- Локальное состояние через useState
- Формы с контролируемыми компонентами

## Инструкция по запуску

1. Перейдите в директорию `src/`:

   ```bash
   cd src
   ```

2. Установите зависимости:

   ```bash
   npm install
   ```

3. Запустите проект в режиме разработки:

   ```bash
   npm run dev
   ```

4. Откройте браузер по адресу `http://localhost:5173`

5. Для сборки проекта:

   ```bash
   npm run build
   ```

## Вывод

В ходе выполнения лабораторной работы был разработан портал стажировок на React.
