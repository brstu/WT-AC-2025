### 2. Запуск проекта

#### 2.1 Установка зависимостей

* Откройте проект в VSCode.
* Установите зависимости:

```bash
npm install
```

#### 2.2 Запуск сервера json-server

* Запустить сервер для работы с данными:

```bash
npx json-server db.json --port 3000
```

* Запустить React-приложение:

```bash
npm run dev
```

* Сервер React доступен по адресу:

```bash
http://localhost:5173
```

* API доступно по адресу:

```bash
http://localhost:3000/recipes
```
