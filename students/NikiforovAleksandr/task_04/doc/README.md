# Лабораторная работа 04. Вариант №18 — Каталог артов (SPA)

## Описание

Одностраничное приложение на чистом JS без фреймворков.
Каталог артов с CRUD, поиском и фильтрацией по тегам. Данные хранятся в `localStorage`.

---

## Функциональность

* **CRUD**: создание, просмотр, редактирование, удаление артов
* **Hash-маршрутизация**: `#/items`, `#/items/:id`, `#/new`, `#/items/:id/edit`
* **Поиск и фильтры**: по названию, описанию и тегам; состояние сохраняется в URL
* **UX**:

  * Подтверждение удаления
  * Блокировка кнопки при сохранении
  * Валидация форм
  * Уведомления о результатах операций (`Toast`)
  * Закрытие детального просмотра по Esc
* **Адаптивная верстка**: корректно на мобильных устройствах

---

## Маршруты

| Hash               | Описание                  |
| ------------------ | ------------------------- |
| `#/items`          | Список артов              |
| `#/items/:id`      | Детальная информация      |
| `#/new`            | Форма создания арта       |
| `#/items/:id/edit` | Форма редактирования арта |

---

## API (локальное)

* `api.getList()` — список артов
* `api.getItem(id)` — деталь арта
* `api.create(data)` — создать арт
* `api.update(id, data)` — редактировать арт
* `api.remove(id)` — удалить арт
