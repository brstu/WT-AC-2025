## Цель работы
Создать одностраничный адаптивный сайт на тему «Здоровый образ жизни» с использованием современных возможностей HTML5, CSS3 и JavaScript. Реализовать семантическую разметку, адаптивную вёрстку, доступность, интерактивные элементы и тёмную тему.

## Структура проекта

- index.html — основная разметка(header, nav, main, section, footer).
- styles.css — стили с поддержкой тёмной темы,  адаптивности.
- script.js — JavaScript для переключения темы и трекера привычек.
- readme.md - отчёт
  - 4.jpg (Спортсмен)
  - 3.jpg (салат с курицей и авокадо)
  - 2.jpg (куриные грудки на гриле)
  - img/ (скриншоты)

## Реализованные элементы
- Семантическая разметка HTML5 (`header`, `nav`, `main`, `section`, `article`, `footer`, `table`)
- Якорная навигация по секциям
- CSS Grid + 2 медиазапроса (≤768 px и ≥1024 px)
- Форма + динамическая таблица привычек на JavaScript
- Полная таблица «График тренировок» на 7 дней
- Оптимизация изображений (`loading="lazy"`)
- Полностью переключаемая тёмная тема с сохранением в `localStorage`
- Автоматическая тёмная тема через `prefers-color-scheme`
- Skip-link для клавиатурной навигации
- Видимые `:focus`-стили у всех элементов
- ARIA-атрибуты (`aria-labelledby`, `aria-label`, `aria-live="polite"`)
- Контрастность ≥ 4.5:1 в обеих темах

## Архитектура вёрстки

- **CSS Grid**:
  - `.container`: `repeat(auto-fit, minmax(300px, 1fr))` для адаптивной сетки секций.
  - `.grid`: `grid-template-columns: auto 1fr` для выравнивания изображений и текста.
- **Медиазапросы**:
  - `@media (max-width: 768px)`: для мобильных устройств (навигация блочная, `.grid` в одну колонку).
  - `@media (min-width: 1024px)`: для десктопов (увеличение ширины контейнера, отступов, размеров изображений).
- **Тёмная тема**:
  - Реализована через CSS-переменные и атрибут `data-theme`.
  - Поддержка `prefers-color-scheme` для автоматического переключения.
  - Сохранение выбора темы в `localStorage`
