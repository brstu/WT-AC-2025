# Отчёт ЛР№1

Твой путь к здоровому образу жизни - Одностраничный сайт

**Описание проекта**
Одностраничный сайт "Твой путь к здоровому образу жизни" посвящён здоровому питанию, тренировкам и отслеживанию привычек. Реализован с акцентом на семантику HTML5, адаптивность и доступность.

## Структура проекта
- index.html — основная разметка(header, nav, main, section, footer).
- styles.css — стили с поддержкой тёмной темы и адаптивности.
- script.js — JavaScript для переключения темы и трекера привычек.
  - 4.jpg (Спортсмен)
  - 3.jpg (салат с курицей и авокадо)
  - 2.jpg (куриные грудки на гриле)

## Архитектура вёрстки

- **CSS Grid**:
  - `.container`: `repeat(auto-fit, minmax(300px, 1fr))` для адаптивной сетки секций.
  - `.grid`: `grid-template-columns: auto 1fr` для выравнивания изображений и текста.
- **Медиазапросы**:
  - `@media (max-width: 768px)`: для мобильных устройств (навигация блочная, `.grid` в одну колонку).
  - `@media (min-width: 1024px)`: для десктопов (увеличение ширины контейнера, отступов, размеров изображений).
- **Тёмная тема**:
  - Реализована через CSS-переменные и атрибут `data-theme`.
  - Поддержка `prefers-color-scheme` для автоматического переключения.
  - Сохранение выбора темы в `localStorage`

### Зоны Flexbox

Навигация (nav): Flexbox с flex-direction: column для мобильных устройств, переключается на row для планшетов и десктопов.

### Зоны CSS Grid

- Flexbox не используется, так как CSS Grid полностью обеспечивает адаптивность.

### Медиазапросы

- ≤768px: Мобильная вёрстка, одноколоночный layout для `.grid`, навигация (`nav ul li`) становится блочной, уменьшены отступы.
- ≥1024px: Десктопная вёрстка, увеличена ширина контейнера (`max-width: 1400px`), увеличены отступы и размеры изображений в `.grid`.
