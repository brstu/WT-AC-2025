## Цель работы
Освоить современные механизмы асинхронного программирования в JavaScript (`async/await`, `AbortController`, `fetch`), реализовать клиентское HTTP-кэширование с поддержкой ETag и 304 Not Modified, а также применить лучшие практики UX: отмена запросов, дебаунс, skeleton-загрузчики, бесконечная подгрузка.

### 2. Реализованные элементы
- `async/await` + `try/catch/finally`
- Кастомный `fetchWithRetry` с экспоненциальным backoff, таймаутом и повторными попытками
- Отмена предыдущих запросов через `AbortController`
- In-memory кэш на основе `Map` с TTL = 5 минут и поддержкой ETag / 304 Not Modified
- Принудительное обновление по кнопке «Обновить»
- Дебаунс ввода поиска (400 мс)
- Бесконечная прокрутка (IntersectionObserver)
- Skeleton-загрузчики и статус-бар
- Состояния: loading, error, empty, «из кэша», «304 Not Modified»
- Полностью неблокирующий UI
- Современный адаптивный дизайн с поддержкой тёмной темы (`prefers-color-scheme`)
- Высокие показатели Lighthouse (Performance ≥ 95, Accessibility 100)
