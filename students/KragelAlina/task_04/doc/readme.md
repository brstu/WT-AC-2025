## Цель работы

- Реализовать простой SPA без фреймворков с маршрутизацией и состояниями загрузки/ошибок.
- Выполнить CRUD к учебному API (реальному или mock).

---

### Вариант №10

Энциклопедия аниме: список тайтлов, страница тайтла, форма добавления/редактирования

## Функциональность

- Полный CRUD (создание, просмотр, редактирование, удаление)  
- Состояния загрузки, пустого списка и ошибок  
- Уведомления о действиях (toast)  
- Тёмная / светлая тема с сохранением в localStorage  
- Адаптивная вёрстка (мобильные + десктоп)  
- Floating-labels в формах  
- Семантически правильная разметка и модульной архитектуры
