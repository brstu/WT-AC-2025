# Каталог художественных работ

Веб-приложение для просмотра и фильтрации художественных работ.

## Технологии

- HTML/CSS/JavaScript
- Node.js + Express
- Jest для тестирования
- Docker
- GitHub Actions

## Запуск

### Локально

```bash
npm install
npm start
```

### Docker

```bash
docker-compose up
```

## Тесты

```bash
npm test
```
