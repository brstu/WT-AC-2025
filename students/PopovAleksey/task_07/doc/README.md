## Ход выполнения работы

### 1. Структура проекта

Проект разделен на две основные директории:

- **doc/** — документация и скриншоты
  - `README.md` — отчет по работе
  - `screenshots/` — скриншоты приложения

- **src/** — исходный код приложения
  - `App.jsx` — главный компонент с роутингом и всеми страницами
  - `main.jsx` — точка входа в приложение
  - `index.html` — HTML шаблон
  - `package.json` — зависимости проекта
  - `vite.config.js` — конфигурация Vite
  - `.env` — переменные окружения
  - `App.test.jsx` — тесты компонентов

### 2. Реализованные элементы

В рамках лабораторной работы были реализованы следующие функции:

**Страницы:**

- Главная страница со списком художественных работ
- Детальный просмотр отдельной работы
- Страница создания новой работы
- Страница редактирования существующей работы
- Страница 404 для несуществующих маршрутов

**Маршрутизация:**

- React Router v6
- Навигация между страницами
- Динамические маршруты для просмотра и редактирования

**Работа с данными:**

- Mock API для эмуляции серверных запросов
- Получение списка работ
- Просмотр деталей работы
- Создание новых работ
- Редактирование существующих работ
- Удаление работ
- Обработка состояний loading/error

**Формы:**

- React Hook Form для управления формами
- Базовая валидация полей (обязательные поля, диапазоны)
- Отображение ошибок валидации

**Управление состоянием:**

- useState для локального состояния компонентов
- useEffect для загрузки данных

**Технологии:**

- Vite как сборщик
- React 18
- React Router DOM v6
- React Hook Form
- Переменные окружения (.env)

## Инструкция по запуску

1. Перейти в директорию `src/`:

   ```bash
   cd src
   ```

2. Установить зависимости:

   ```bash
   npm install
   ```

3. Запустить в режиме разработки:

   ```bash
   npm run dev
   ```

4. Открыть в браузере: `http://localhost:5173`

5. Запустить тесты:

   ```bash
   npm test
   ```

6. Собрать для продакшена:

   ```bash
   npm run build
   ```
