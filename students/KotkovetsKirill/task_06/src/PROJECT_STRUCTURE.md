```

│

│   └── screenshots/              # Папка для скриншотов
│
└── src/                          # Исходный код
    ├── prisma/                   # Prisma ORM
    │   └── schema.prisma         # Схема базы данных
    │
    ├── screenshots-html/         # HTML страницы для скриншотов
    │   ├── signup.html           # Страница регистрации
    │   ├── login.html            # Страница входа
    │   ├── create.html           # Создание записи
    │   └── list.html             # Список записей
    │
    ├── index.js                  # Главный файл сервера
    ├── seed.js                   # Заполнение БД тестовыми данными
    ├── demo.html                 # Демо-страница для тестирования API
    │
    ├── package.json              # Зависимости проекта
    ├── .env                      # Переменные окружения (не коммитить)

    ├── .gitignore                # Игнорируемые файлы
    │
    ├── setup.bat                 # Скрипт автоматической установки (Windows)
    ├── start.bat                 # Скрипт запуска сервера (Windows)
    │
    ├── README.md                 # Краткая документация проекта
    ├── INSTALL.md                # Подробная инструкция по установке
    └── NOTES.md                  # Известные особенности реализации
```

## Описание файлов

### Основные файлы

- **index.js** - Главный файл сервера с Express, содержит все роуты и логику
- **seed.js** - Скрипт для заполнения базы данных тестовыми пользователями и записями
- **schema.prisma** - Схема базы данных с моделями User и HealthRecord

### Конфигурация

- **package.json** - Зависимости: express, prisma, bcrypt, jsonwebtoken, cors, dotenv
- **.env** - Переменные окружения: DATABASE_URL, JWT_SECRET, PORT
- **.gitignore** - Исключает node_modules, .env, *.db

### Документация

- **doc/README.md** - Полный отчет по лабораторной работе
- **INSTALL.md** - Инструкция по установке и запуску
- **README.md** - Краткая справка по API
- **NOTES.md** - Известные особенности реализации

### Утилиты

- **demo.html** - Интерактивная HTML-страница для тестирования всех эндпоинтов API
- **setup.bat** - Автоматическая установка для Windows
- **start.bat** - Быстрый запуск сервера для Windows

### Демо-страницы

- **screenshots-html/** - HTML-страницы для демонстрации интерфейсов

## API Эндпоинты

### Авторизация (публичные)

- `POST /api/auth/signup` - Регистрация
- `POST /api/auth/login` - Вход

### Записи здоровья (требуется авторизация)

- `GET /api/records` - Получить все записи
- `POST /api/records` - Создать запись
- `GET /api/records/:id` - Получить запись
- `PUT /api/records/:id` - Обновить запись
- `DELETE /api/records/:id` - Удалить запись

### Тестовый

- `GET /api/test` - Проверка работы сервера

## Технологии

- **Node.js** + **Express** - Серверный фреймворк
- **Prisma ORM** - Работа с базой данных
- **SQLite** - База данных
- **JWT** - Токены авторизации
- **bcrypt** - Хеширование паролей
- **CORS** - Разрешение кросс-доменных запросов
- **dotenv** - Управление переменными окружения
