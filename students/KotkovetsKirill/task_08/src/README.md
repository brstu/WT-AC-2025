# Coffee Shop Catalog

Справочник кофеен - веб-приложение для поиска и просмотра информации о кофейнях.

## Запуск проекта

### Локальная разработка

```bash
npm install
npm run dev
```

### Docker

```bash
docker-compose up --build
```

## Тестирование

```bash
# Unit тесты
npm test

# E2E тесты
npm run test:e2e
```

## Структура

- `/src` - исходный код
- `/components` - React компоненты
- `/e2e` - e2e тесты
