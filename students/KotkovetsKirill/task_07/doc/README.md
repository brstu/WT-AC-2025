## Цель работы

Создать полноценный SPA на React с маршрутизацией, управлением состоянием, формами и взаимодействием с данными.

---

### Вариант №35

Справочник кофеен с отзывами и рейтингами.

## Ход выполнения работы

### 1. Структура проекта

```
src/
├── package.json          # Зависимости проекта
├── vite.config.js        # Конфигурация Vite
├── .env                  # Переменные окружения
├── index.html            # HTML-шаблон
├── main.jsx              # Точка входа приложения
├── App.jsx               # Главный компонент с роутингом
├── style.css             # Глобальные стили
├── CoffeeShopsList.jsx   # Страница списка кофеен
├── CoffeeShopDetail.jsx  # Страница детальной информации
├── CoffeeShopNew.jsx     # Страница добавления кофейни
├── CoffeeShopEdit.jsx    # Страница редактирования
└── NotFound.jsx          # Страница 404

doc/
├── README.md             # Отчет по работе
└── screenshots/          # Скриншоты приложения
```

### 2. Реализованные элементы

- **React Router v6**: настроена маршрутизация со всеми необходимыми страницами
- **CRUD операции**: создание, чтение, обновление и удаление кофеен
- **Формы**: формы добавления и редактирования кофеен
- **Состояние**: управление состоянием через useState
- **Отзывы и рейтинги**: возможность просмотра и добавления отзывов
- **Адаптивный дизайн**: базовая стилизация интерфейса
- **Изображения**: интеграция изображений через Unsplash API

## Инструкция по запуску

1. Перейти в директорию `src/`:

```bash
cd src
```

1. Установить зависимости:

```bash
npm install
```

1. Запустить проект в режиме разработки:

```bash
npm run dev
```

1. Открыть браузер по адресу `http://localhost:5173`

1. Для сборки production версии:

```bash
npm run build
```

## Вывод

В ходе выполнения лабораторной работы был создан SPA-справочник кофеен на React. Реализована базовая маршрутизация с использованием React Router v6, включающая страницы списка, детальной информации, добавления, редактирования и 404. Добавлен функционал CRUD-операций для управления кофейнями и система отзывов с рейтингами.

Управление состоянием реализовано через useState. Созданы формы для добавления и редактирования данных. Приложение использует изображения из открытого API Unsplash для визуального оформления карточек кофеен.

Освоены навыки работы с React Router, управления состоянием компонентов, работы с формами и обработки пользовательского ввода.
