## Архитектура вёрстки

### Медиазапросы

- Mobile-first подход
- 2 медиазапроса: `max-width: 600px` и `min-width: 1024px`

### Использование Grid

На десктопе (≥1024px) контейнер `main` использует CSS Grid с двумя колонками для размещения секций.

### Семантика

Базовые HTML5 элементы: `header`, `nav`, `main`, `section`, `footer`.
