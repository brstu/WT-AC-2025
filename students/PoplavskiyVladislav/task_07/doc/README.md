# Каталог мероприятий с заявками

SPA-приложение для управления каталогом мероприятий с системой заявок на участие.
Вариант - 17.
Выполнил - Поплавский Владсилав.

## Функциональность

- Просмотр списка мероприятий с фильтрацией и поиском
- Детальная информация о мероприятии
- Создание и редактирование мероприятий
- Система заявок на участие
- Удаление мероприятий и заявок
- Адаптивный дизайн
- Уведомления об операциях

## Технологии

- React 18
- Vite (сборка)
- Redux Toolkit (управление состоянием)
- React Router v6 (маршрутизация)
- Нативная валидация форм (без внешних библиотек)
- CSS Modules (стилизация)
- JSON Server (mock API)

## Запуск проекта

### 1. Клонирование и установка

```bash
git clone <repository-url>
cd event-catalog
npm install
```

### 2. Запуск сервера разработки

В двух терминалах:

```bash

# Терминал 1: Запуск JSON Server
npx json-server --watch db.json --port 3001

# Терминал 2: Запуск приложения
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000
API будет доступно по адресу: http://localhost:3001

### 3. Сборка для продакшена

```bash
npm run build
npm run preview
```

## API Endpoints

GET /events - получить список мероприятий
GET /events/:id - получить мероприятие по ID
POST /events - создать мероприятие
PUT /events/:id - обновить мероприятие
DELETE /events/:id - удалить мероприятие
GET /applications?eventId=:id - получить заявки для мероприятия
POST /applications - создать заявку
DELETE /applications/:id - удалить заявку
