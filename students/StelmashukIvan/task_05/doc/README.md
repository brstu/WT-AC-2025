# REST API для управления учебными группами и домашними заданиями

Вариант - 20.
Выполнил - Стельмашук Иван.

RESTful API для управления учебными группами и домашними заданиями с валидацией, обработкой ошибок и документацией Swagger.

## Запуск проекта

### Требования

* Node.js 16+
* npm

### Установка

npm install

### Запуск в режиме разработки

npm run dev

### Запуск в production

npm start

### Переменные окружения

Создайте файл .env в корне проекта:

PORT=3000
NODE_ENV=development

## API Endpoints

### Academic Groups (/api/groups)

* GET /api/groups - получить список групп (поддерживает поиск и пагинацию)
* GET /api/groups/:id - получить группу по ID
* POST /api/groups - создать новую группу
* PATCH /api/groups/:id - обновить группу
* DELETE /api/groups/:id - удалить группу

### Assignments (/api/assignments)

* GET /api/assignments - получить список заданий (поддерживает поиск и пагинацию)
* GET /api/assignments/:id - получить задание по ID
* POST /api/assignments - создать новое задание
* PATCH /api/assignments/:id - обновить задание
* DELETE /api/assignments/:id - удалить задание
* GET /api/assignments/group/:groupId - получить задания по группе

### Health Check

* GET /health - проверка работоспособности сервера

## Документация API

* Документация Swagger доступна по адресу: /docs
* Валидация данных
* Группы (Academic Group)

* name: строка, 1-50 символов
* course: число, 1-6
* faculty: строка, 1-100 символов
* studentCount: число, 0-100 (опционально)

### Задания (Assignment)

* title: строка, 1-200 символов
* description: строка, до 1000 символов (опционально)
* dueDate: строка в формате ISO 8601
* maxScore: число, 1-100 (опционально)
* groupId: UUID существующей группы
* subject: строка, 1-100 символов

## Коды ответов

* 200 - успешный запрос
* 201 - ресурс создан
* 204 - ресурс удален
* 400 - неверный запрос
* 404 - ресурс не найден
* 422 - ошибка валидации
* 500 - внутренняя ошибка сервера

## Хранение данных

Данные хранятся в памяти (in-memory storage) и сбрасываются при перезапуске сервера.
