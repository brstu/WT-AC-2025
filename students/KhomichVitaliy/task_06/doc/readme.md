## Возможности

- Вход / Регистрация
- Восстановление пароля
- Разместить вакансию
- Просмотр всех вакансий
- Просмотр деталей вакансии
- Добавление вакансий в закладки
- Отслеживание вакансий в закладках в виде таблицы
- Отслеживание вакансий, на которые вы откликнулись, в виде таблицы
- Отслеживание опубликованных вакансий и просмотр поданных заявок
- Принятие и отклонение различных заявок
- Обновление статуса откликов на вакансию в зависимости от действий работодателя
- JWT‑валидация для каждого авторизованного запроса

#### (Следующая функция работает только в локальной среде, так как Vercel не позволяет операции записи в бесплатном плане, поэтому резюме не сохраняются в продакшене. Можно использовать Firebase или AWS S3 для хранения, но локально всё работает корректно)
- Просмотр или скачивание резюме кандидатов

## Технологии

- Next.js  
- Tailwind CSS  
- Redux Toolkit  
- Joi Validation  
- MongoDB  
- SWR hooks для получения данных из API  

---

## Переменные окружения

Для запуска проекта необходимо добавить следующие переменные в файл `.env`:

`DB_URI` = ваш URL MongoDB  

`JWT_SECREAT` = ваш секретный ключ JWT  

`NEXT_PUBLIC_API_BASE_URL` = базовый URL для localhost → http://localhost:3000
