# Блог-платформа

React SPA приложение для управления блогом с возможностью создания, редактирования и удаления постов, а также просмотра и добавления комментариев.

## Технологии

- React 18
- React Router v6
- Vite
- JSONPlaceholder API

## Установка и запуск

1. Установите зависимости:

```bash
npm install
```

1. Запустите dev сервер:

```bash
npm run dev
```

1. Соберите проект:

```bash
npm run build
```

## Структура проекта

```text
src/
├── pages/          # Страницы приложения
│   ├── PostsList.jsx     # Список постов
│   ├── PostDetail.jsx    # Детальный просмотр поста
│   ├── CreatePost.jsx    # Создание нового поста
│   ├── EditPost.jsx      # Редактирование поста
│   └── NotFound.jsx      # Страница 404
├── App.jsx         # Главный компонент с роутингом
├── main.jsx        # Точка входа
└── index.css       # Глобальные стили
```

## Функциональность

- Просмотр списка всех постов
- Просмотр деталей поста с комментариями
- Создание нового поста
- Редактирование существующего поста
- Удаление поста
- Добавление комментариев к посту
- Обработка состояний загрузки и ошибок
- 404 страница для несуществующих маршрутов

## API

Проект использует [JSONPlaceholder](https://jsonplaceholder.typicode.com/) - бесплатный REST API для тестирования и прототипирования.

Endpoints:

- GET /posts - получение всех постов
- GET /posts/:id - получение поста по ID
- GET /posts/:id/comments - получение комментариев к посту
- POST /posts - создание нового поста
- PUT /posts/:id - обновление поста
- DELETE /posts/:id - удаление поста
- POST /comments - добавление комментария
