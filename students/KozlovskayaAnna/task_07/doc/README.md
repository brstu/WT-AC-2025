### 2. Реализованные элементы

- **Страницы**: список постов, деталь поста, создание нового поста, редактирование поста, страница 404
- **Маршрутизация**: React Router v6 для навигации между страницами
- **CRUD операции**: получение списка постов, просмотр деталей, создание, редактирование и удаление постов
- **Работа с API**: использование JSONPlaceholder API для получения и отправки данных
- **Формы**: формы создания и редактирования постов с базовой валидацией
- **Комментарии**: отображение комментариев к постам и возможность добавления новых
- **Состояния**: обработка loading состояния, отображение ошибок
- **Уведомления**: использование alert для уведомлений пользователя

## Инструкция по запуску

1. Перейти в папку `src`:

```bash
cd src
```

1. Установить зависимости:

```bash
npm install
```

1. Запустить проект в режиме разработки:

```bash
npm run dev
```

1. Открыть в браузере по адресу, указанному в консоли (обычно <http://localhost:5173>)

## Вывод

В ходе выполнения лабораторной работы был создан SPA на React с использованием Vite. Реализована маршрутизация с помощью React Router v6, созданы страницы для просмотра списка постов, детального просмотра, создания и редактирования постов. Выполнена интеграция с JSONPlaceholder API для получения и отправки данных. Реализованы формы с базовой валидацией. Освоены навыки работы с React hooks (useState, useEffect), управления состоянием приложения и работы с внешним API.
