# Modern Fashion Trends - Лабораторная работа 01

Адаптивный семантический сайт о современных трендах в моде и стиле с акцентом на доступность (a11y).

## Демо

[Ссылка на GitHub Pages](https://ваш-username.github.io/lab01-fashion-trends/)

## Технологии

- HTML5 (семантическая разметка)
- CSS3 (Flexbox, Grid, CSS Custom Properties)
- Vanilla JavaScript
- Mobile-first подход

## Особенности реализации

### Семантика и структура
- Использование HTML5 семантических тегов (`header`, `nav`, `main`, `section`, `article`, `footer`)
- Правильная иерархия заголовков (`h1-h4`)
- ARIA-атрибуты для улучшения доступности
- Логическая структура документа

### Адаптивный дизайн
- **Mobile-first подход**: базовые стили для мобильных устройств
- **2 брейкпоинта**:
  - 601px: планшетная версия
  - 1025px: десктопная версия
- **Адаптивные сетки**: CSS Grid и Flexbox
- **Адаптивные изображения**: lazy loading, `srcset`

### Доступность (a11y)
- Контрастность не менее 4.5:1 (проверено через WebAIM Contrast Checker)
- Полная клавиатурная навигация
- Видимый фокус для всех интерактивных элементов
- Alt-тексты для всех изображений
- ARIA-label для интерактивных элементов
- Skip link для пропуска навигации
- Поддержка screen readers

### Качество кода
- Валидный HTML/CSS (W3C Validator)
- Высокие показатели Lighthouse
- Чистая и понятная структура кода
- Комментарии в коде
