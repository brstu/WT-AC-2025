### 1. Структура проекта

Проект организован следующим образом:

```
task_07/
├── doc/
│   ├── README.md
│   └── screenshots/
│       ├── screenshot1_home.txt

│       └── screenshot5_form.txt

    ├── pages/
    │   ├── Home.jsx
    │   ├── List.jsx
    │   ├── Detail.jsx
    │   ├── New.jsx
    │   ├── Edit.jsx

    ├── App.jsx
    ├── main.jsx
    ├── index.html
    ├── vite.config.js
```

**Основные файлы:**

- `index.html` — главная HTML-страница
- `main.jsx` — точка входа React-приложения
- `App.jsx` — корневой компонент с настройкой маршрутизации
- `pages/` — директория со всеми страницами приложения
- `vite.config.js` — конфигурация сборщика Vite
- `package.json` — зависимости проекта
- `.env` — переменные окружения для API

### 2. Реализованные элементы

**Реализованная функциональность:**

- **Маршрутизация:** React Router v6 с основными маршрутами (список, деталь, создание, редактирование, 404)
- **Страницы:**
  - Главная страница (Home) - приветственная страница
  - Список товаров (List) - отображение всех гаджетов
  - Детальная страница (Detail) - подробная информация о товаре
  - Создание нового товара (New) - форма добавления
  - Редактирование товара (Edit) - форма изменения
  - Страница 404 (NotFound) - обработка несуществующих маршрутов

- **Работа с API:**
  - Получение списка товаров (GET)
  - Получение деталей товара (GET)
  - Создание товара (POST)
  - Обновление товара (PUT)
  - Удаление товара (DELETE)

- **Фильтрация данных:**
  - Фильтр по категориям
  - Фильтр по цене

- **Формы:**
  - Форма добавления нового товара
  - Форма редактирования существующего товара
  - Базовая валидация (обязательные поля)

- **Состояния:**
  - Loading (загрузка)
  - Отображение данных
  - Уведомления об успешных операциях

## Инструкция по запуску

1. Перейти в директорию `src`:

   ```bash
   cd src
   ```

2. Установить зависимости:

   ```bash
   npm install
   ```

3. Запустить dev-сервер:

   ```bash
   npm run dev
   ```

4. Открыть приложение в браузере по адресу: `http://localhost:5173`

## Используемые технологии

- **React 18** - библиотека для создания пользовательских интерфейсов
- **React Router v6** - маршрутизация
- **Vite** - сборщик и dev-сервер
- **FakeStore API** - тестовое API для работы с данными
