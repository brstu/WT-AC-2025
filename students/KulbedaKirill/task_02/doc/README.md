## Цель работы

Освоить работу с DOM: добавление, удаление и изменение элементов интерфейса. Научиться обрабатывать события, использовать делегирование, работать с клавиатурной доступностью. Реализовать клиентскую валидацию форм.

---

### Вариант №12

Магазин мерча (front-demo): список карточек, делегирование на «в корзину», модалка корзины.

## Ход выполнения работы

### 1. Структура проекта

```text
+---doc
|   |   README.md
|   |
|   \---screenshots
|           ...
|
\---src
    |   index.html
    |   styles.css
    |   main.js
    |
    \---img
            ...
```

- `index.html` — основная страница магазина мерча
- `styles.css` — стилизация и медиазапросы
- `main.js` — JavaScript логика компонентов

### 2. Реализованные элементы

- Карточки товаров с кнопками "В корзину" и "Лайк"
- Делегирование событий на контейнере товаров
- Модальное окно корзины с возможностью удаления товаров
- Аккордеон в разделе "О нас"
- Бургер-меню для мобильных устройств
- Форма обратной связи с валидацией
- Валидация при вводе и при отправке
- Адаптивная верстка с брейкпоинтами
