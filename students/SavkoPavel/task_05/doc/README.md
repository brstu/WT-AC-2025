## Ход выполнения работы

## Цель работы

Целью лабораторной работы является разработка серверного REST API на платформе Node.js с использованием фреймворка Express. В рамках работы необходимо реализовать CRUD-эндпоинты для выбранного ресурса, добавить валидацию входных данных, централизованную обработку ошибок и автоматическую документацию API с помощью Swagger (OpenAPI).

---

## Описание проекта

В рамках лабораторной работы был разработан REST API «Доска объявлений», предназначенный для управления объявлениями. API предоставляет возможности создания, получения, обновления и удаления объявлений. Для удобства тестирования и использования реализована интерактивная документация Swagger UI.

---

## Архитектура проекта

### Структура

- **Routes** ― описание HTTP-маршрутов и Swagger-аннотаций\
- **Controllers** ― бизнес-логика обработки запросов\
- **Schemas** ― схемы валидации данных (Zod)\
- **Middlewares** ― валидация, логирование, обработка ошибок\
- **Swagger** ― генерация OpenAPI-документации\

---

## Реализованный функционал (CRUD)

| Метод | Эндпоинт | Описание |
|------------------------------------------------------|-----------|
| GET | /api/ads | Получение списка объявлений |
| GET | /api/ads/{id} | Получение объявления по ID |
| POST | /api/ads | Создание нового объявления |
| PUT | /api/ads/{id} | Обновление объявления |
| DELETE | /api/ads/{id} | Удаление объявления |
