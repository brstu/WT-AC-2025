## Ход выполнения работы

### 1. Структура проекта

Проект организован следующим образом:

```
task_07/
├── index.html
├── package.json
├── vite.config.js
├── .env
├── README.md
└── src/
    ├── main.jsx
    ├── App.jsx
    ├── api.js
    ├── router.jsx
    └── pages/
        ├── List.jsx
        ├── Detail.jsx
        ├── Create.jsx
        ├── Edit.jsx
        └── NotFound.jsx
```

**Основные файлы:**

index.html — входная HTML-страница

main.jsx — точка входа React-приложения

App.jsx — основной layout с навигацией

router.jsx — конфигурация маршрутов React Router

api.js — функции для работы с серверным API

pages/ — компоненты страниц приложения

.env — переменные окружения (URL API)

### 2. Реализованные элементы

**Основной функционал:**

- **Маршрутизация с React Router v6:**
    В приложении реализованы следующие маршруты:

    / — список стартапов

    /items/:id — детальная информация о стартапе

    /new — создание нового стартапа

    /edit/:id — редактирование стартапа

    "*" — страница 404

- **Работа с данными и API:**

  Взаимодействие с серверным API из предыдущих лабораторных работ
  Получение списка и детальной информации
  Создание, редактирование и удаление записей (CRUD)
  Хранение базового URL API в .env (VITE_API_URL)

- **Управление состоянием:**
  Использована библиотека TanStack Query
  useQuery — загрузка списка и деталей
  useMutation — создание, обновление и удаление
  Реализованы состояния:
    loading
    error
    empty

- **Формы:**
  Форма создания стартапа
  Форма редактирования стартапа
  Базовая валидация обязательных полей (проверка на пустые значения)
  Управление состоянием форм через useState

- **Интерфейс:**
  Простая навигация
  Кнопки действий (создать, редактировать, удалить)
  Минимальный, но понятный пользовательский интерфейс

## Запуск проекта

1. Перейти в директорию `src`:

```bash
cd src
```

1. Установить зависимости:

```bash
npm install
```

1. Запустить проект в режиме разработки:

```bash
npm run dev
```

1. Открыть браузер по адресу: `http://localhost:5173`

## Сборка для продакшена

```bash
npm run build
```

## Вывод

В ходе выполнения лабораторной работы был создан полноценный SPA каталог питомцев из приюта на React. Реализована маршрутизация с использованием React Router v6, включая вложенные маршруты и защищенные маршруты для авторизованных пользователей.

Освоены навыки работы с формами, валидацией данных, управлением состоянием через React hooks (useState, useEffect). Реализован полный цикл CRUD операций с данными, хранящимися в localStorage.

Приложение имеет интуитивно понятный интерфейс с навигацией, фильтрацией данных и уведомлениями о выполненных операциях. Использованы открытые API для получения изображений животных, что делает каталог более наглядным и привлекательным.
