## Тема

HTML/CSS: семантика, адаптивность и доступность

## Цель работы

Освоить основы семантической вёрстки HTML5, реализовать адаптивную структуру веб-страницы по принципу mobile-first и обеспечить её доступность в соответствии с современными рекомендациями.

## Содержание репозитория

- `index.html` — семантическая одностраничная страница (header, nav, main, sections, footer).
- `styles.css` — mobile-first стили с breakpoints на 600px и 1024px; поддержка светлой/тёмной темы.
- `assets/` — SVG-иконки и фоновые изображения (earth.svg, shop.svg, repeat.svg, light.svg, fon.jpg).

## Как запустить локально

1. Открыть папку проекта в терминале.
2. Запустить простой статический сервер (пример для PowerShell):

```powershell
python -m http.server 5000
# или
npx serve -s . -l 5000
```

1. Открыть в браузере: <`http://localhost:5000`>

## Что реализовано

- Семантическая разметка HTML5: `header`, `nav`, `main`, `section`, `footer`.
- Доступность (a11y): skip-link, `label` для формы, `alt` для изображений, `aria-*` для кнопок, :focus-visible стили.
- Мобильная первая вёрстка с медиазапросами: один/два/три столбца для секции советов на разных ширинах.
- Адаптивная сетка: CSS Grid для `tips-grid`, Flexbox для `header-inner`.
- Фиксированный хедер с полупрозрачным фоном и blur, кнопка переключения тёмной темы (сохранение выбора в localStorage).
- Минимальные размеры интерактивных элементов (touch targets ≥44px), lazy-loading для изображений, SVG-иконки с явными размерами.
