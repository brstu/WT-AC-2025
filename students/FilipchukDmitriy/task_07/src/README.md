# Справочник Open-Source проектов

React SPA-приложение для управления справочником open-source проектов.

## Установка и запуск

1. Установите зависимости:

   ```bash
   npm install
   ```

1. Запустите dev-сервер:

   ```bash
   npm run dev
   ```

1. Откройте браузер по адресу: http://localhost:5173

## Сборка для продакшена

```bash
npm run build
```

## Технологии

- React 18.2.0
- React Router DOM 6.20.0
- Vite 5.0.8

## Особенности

- Маршрутизация с React Router v6
- CRUD операции с проектами
- Интеграция с GitHub API
- Хранение данных в localStorage
- Валидация форм
