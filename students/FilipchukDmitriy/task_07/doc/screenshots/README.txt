# Screenshot Placeholders

Скриншоты приложения будут добавлены после запуска проекта.

Для создания скриншотов:
1. Запустить приложение командой `npm run dev` из папки `src/`
2. Сделать скриншоты следующих экранов:
   - main-page.png - главная страница со списком проектов
   - project-detail.png - детальная страница проекта
   - create-project.png - форма создания проекта
   - edit-project.png - форма редактирования проекта

Разрешение скриншотов: 1920x1080 или 1366x768
