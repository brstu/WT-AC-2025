### 2. Реализованные элементы

**Маршрутизация (React Router v6):**
- `/` — главная страница со списком проектов
- `/project/:id` — детальная страница проекта
- `/new` — создание нового проекта
- `/edit/:id` — редактирование проекта
- `*` — страница 404 для несуществующих маршрутов

**Работа с данными:**
- Загрузка списка проектов из GitHub API
- Хранение данных в localStorage
- Создание новых записей
- Редактирование существующих проектов
- Удаление проектов
- Отображение состояний loading и error

**Формы:**
- Форма создания проекта с полями: название, описание, автор, количество звезд, язык программирования, URL
- Форма редактирования с предзаполнением данных
- Базовая валидация обязательных полей
- Валидация типа данных (число для звезд)

**Интерфейс:**
- Навигационное меню
- Карточки проектов в виде сетки
- Детальная информация о проекте
- Кнопки управления (редактировать, удалить)
- Обратная навигация

## Инструкция по запуску

1. Перейти в директорию `src/`:

   ```bash
   cd src
   ```

2. Установить зависимости:

   ```bash
   npm install
   ```

3. Запустить dev-сервер:

   ```bash
   npm run dev
   ```

4. Открыть в браузере адрес, указанный в консоли (обычно `http://localhost:5173`)

## Использованные технологии

- **React** 18.2.0 — библиотека для создания пользовательских интерфейсов
- **React Router DOM** 6.20.0 — маршрутизация в приложении
- **Vite** 5.0.8 — инструмент сборки
- **GitHub API** — источник данных о популярных проектах
- **localStorage** — клиентское хранилище данных
- **Picsum Photos API** — генерация изображений для проектов

## Вывод

В ходе выполнения лабораторной работы был разработан одностраничный React-приложение "Справочник open-source проектов". Реализована маршрутизация с использованием React Router v6, включая основные страницы (список, детали, создание, редактирование) и обработку несуществующих маршрутов (404).

Освоены навыки работы с формами в React, включая управление состоянием формы и валидацию данных. Реализовано взаимодействие с внешним API (GitHub API) для загрузки данных, а также полный CRUD-функционал с хранением данных в localStorage.

Приложение позволяет просматривать список популярных JavaScript-проектов, просматривать детальную информацию, добавлять собственные проекты, редактировать и удалять записи. Реализованы базовые состояния загрузки и обработки ошибок.
