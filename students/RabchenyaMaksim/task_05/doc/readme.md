## **Вариант №39 — Музыкальные альбомы**

По заданию реализуется серверное API с CRUD-функциями для коллекции музыкальных альбомов.

## **2. Реализованный функционал**

### **2.1 Конфигурация Express**

- подключены middleware: `cors`, `express.json()`, `morgan`;
- сервер запускается на порту из `.env` или `3000`;
- Swagger подключён на `/docs`.

---

### **2.2 REST API для ресурса "albums"**

| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/api/v1/albums` | Список альбомов + поиск + пагинация |
| GET | `/api/v1/albums/:id` | Получить альбом по ID |
| POST | `/api/v1/albums` | Создать новый альбом |
| PUT | `/api/v1/albums/:id` | Изменить альбом |
| PATCH | `/api/v1/albums/:id` | Частичное обновление |
| DELETE | `/api/v1/albums/:id` | Удалить альбом |

Поддерживаемые параметры:

- `q` — поиск по полям title и artist
- `page`, `limit` — пагинация
