## Цель работы

Целью лабораторной работы является освоение разработки одностраничных веб-приложений (SPA) с использованием библиотеки React, а также изучение принципов маршрутизации, управления состоянием и работы с REST API.

## Ход выполнения работы

### 1. Структура проекта

В ходе выполнения лабораторной работы был разработан React‑проект со следующей структурой:

- `index.html` — HTML‑шаблон приложения
- `vite.config.ts` — конфигурация сборщика Vite
- `db.json` — файл mock‑данных для json‑server
- `src/` — исходный код приложения
  - `app/` — конфигурация Redux Store и маршрутизации
  - `features/` — бизнес‑логика (турниры, матчи)
  - `pages/` — страницы приложения
  - `shared/` — переиспользуемые компоненты
  - `main.tsx` — точка входа приложения
  - `index.css` — глобальные стили

### 2. Реализованные элементы

В рамках лабораторной работы реализованы следующие элементы:

- Одностраничное приложение (SPA) на React
- Маршрутизация страниц с использованием React Router
- Управление состоянием с помощью Redux Toolkit
- Работа с REST API через RTK Query
- CRUD‑операции для сущности «Турнир»
- Форма добавления и редактирования турниров
- Асинхронное получение данных
- Использование mock‑сервера json‑server

### 3. Скриншоты выполненной лабораторной работы

В ходе работы были получены следующие результаты:
- Главная страница со списком турниров
- Страница добавления турнира
- Страница редактирования турнира
- Удаление турнира
- Работа REST API (`/tournaments`, `/matches`)
