# JobHost — Портал вакансий и поиска работы

Полноценное веб-приложение для поиска работы и найма персонала.  
Проект представляет собой REST-клиент в виде Single Page Application (SPA) с серверной частью, аутентификацией и работой с базой данных.

## Обзор проекта

JobHost соединяет соискателей и работодателей в единой платформе.  
Соискатели могут создавать профили, загружать резюме и откликаться на вакансии.  
Рекрутеры публикуют вакансии, просматривают отклики и управляют кандидатами.

Приложение построено на современном стеке и ориентировано на удобство, безопасность и производительность.

## Основные возможности

### Для соискателей

- Регистрация и создание персонального профиля
- Загрузка и управление резюме
- Поиск вакансий с фильтрами (категория, город, уровень опыта, зарплата)
- Подача заявок на вакансии
- Отслеживание статуса своих заявок

### Для работодателей/рекрутеров

- Публикация и редактирование вакансий
- Просмотр поступивших откликов
- Фильтрация и поиск кандидатов
- Управление статусами заявок (приглашение, отказ и т.д.)

### Общие функции

- Безопасная аутентификация и авторизация
- Адаптивный интерфейс (desktop + mobile)
- Персональные дашборды для разных ролей
- Уведомления о важных событиях

## Технологический стек

### Frontend

- Next.js (App Router)
- React
- Tailwind CSS
- shadcn/ui — набор доступных и кастомизируемых компонентов

### Backend

- Node.js
- Express.js
- Prisma ORM

### База данных

- PostgreSQL

### Аутентификация

- NextAuth.js
- JWT-токены
- Хэширование паролей (bcrypt)

### Дополнительно

- TypeScript (опционально, для типобезопасности)
- ESLint + Prettier для качества кода
