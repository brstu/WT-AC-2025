# Лабораторная работа №6: Базы данных и авторизация (SQLite/PostgreSQL + ORM + JWT)

## Описание проекта

Это серверное приложение на **Node.js** с использованием **Express.js**, реализующее REST API для управления задачами (Tasks) с поддержкой авторизации пользователей на базе **JWT**.

В рамках лабораторной работы:
- Подключена база данных через ORM (Prisma / TypeORM / Sequelize — в примере используется **Prisma**).
- Реализованы сущности **User** и **Task** (с полем `ownerId` для связи с владельцем).
- Перенесён CRUD для задач из предыдущей ЛР в БД.
- Добавлена регистрация и логин пользователей с хешированием паролей (bcrypt).
- Реализована защита маршрутов middleware'ом с проверкой Bearer-токена JWT.
- Доступ к задачам только для их владельца.
- Добавлены меры безопасности: CORS, ограничение размера тела запроса, хранение секретов в `.env`.

## Технологии

- Node.js
- Express.js
- Prisma (ORM) + SQLite / PostgreSQL
- JWT (jsonwebtoken)
- bcrypt (хеширование паролей)
- dotenv (переменные окружения)
- cors, express-rate-limit (по желанию для безопасности)

## Предварительные требования

- Node.js ≥ 18
- npm или yarn
- Для PostgreSQL: установленный сервер (или Docker)
