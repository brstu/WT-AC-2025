# Лабораторная работа 05: REST API на Node.js + Express

**Тема:** Серверное REST API с валидацией, централизованной обработкой ошибок и документацией Swagger.

## Описание проекта

Это реализация простого REST API для управления задачами (**tasks**).  
API поддерживает полный CRUD, валидацию входных данных (с помощью **Zod**), централизованную обработку ошибок и интерактивную документацию через **Swagger UI**.

Данные хранятся временно в памяти (массив в файле).

**Бонусы реализованы:**
- Поиск по тексту (`q`)
- Пагинация с параметрами `limit`/`offset` и метаданными
- Версионирование API (`/api/v1`)

## Технологии

- Node.js ≥ 18
- Express.js
- Zod (валидация)
- swagger-jsdoc + swagger-ui-express (документация OpenAPI)
- dotenv (переменные окружения)
- morgan (логгирование)
- cors
