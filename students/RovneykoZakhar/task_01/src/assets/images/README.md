# Изображения для портфолио

Для корректной работы сайта необходимо добавить изображения портфолио.

## Требуемые изображения

Для каждого элемента портфолио нужны три размера изображения:

1. **portfolio-1-small.jpg** (мобильные устройства, ~400px ширина)
2. **portfolio-1-medium.jpg** (планшеты, ~800px ширина)
3. **portfolio-1-large.jpg** (десктопы, ~1200px ширина)

Повторите для всех 6 элементов портфолио (portfolio-1 до portfolio-6).

## Рекомендации

- Используйте формат JPEG для фотографий
- Оптимизируйте изображения для веба (сжатие без потери качества)
- Сохраняйте соотношение сторон примерно 4:3 или 16:9
- Размеры файлов должны быть оптимизированы для быстрой загрузки

## Примеры тем для изображений

1. Свадебная фотосъёмка — молодожёны в парке
2. Портретная фотосъёмка — женский портрет в студии
3. Семейная фотосъёмка — семья с детьми на природе
4. Корпоративная фотосъёмка — команда в офисе
5. Love Story — пара в городской среде
6. Детская фотосъёмка — ребёнок играет в парке
