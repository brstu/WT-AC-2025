### 2. Реализованные элементы

#### Семантическая разметка HTML5 ✅
- Использованы семантические теги: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`, `<address>`
- Корректная иерархия заголовков: `<h1>` → `<h2>` → `<h3>`
- Логическая структура документа с ARIA landmarks
- Skip link для навигации с клавиатуры

#### Адаптивная вёрстка ✅
- **Mobile-first подход**: базовые стили для мобильных устройств (≤600px)
- **3 брейкпоинта**:
  - Мобильные устройства: ≤600px (одна колонка)
  - Планшеты: 601px - 1024px (две колонки)
  - Десктопы: >1024px (три колонки для портфолио)
- **Flexbox**: используется для навигации, футера и выравнивания элементов
- **CSS Grid**: используется для портфолио, услуг, отзывов и контактов
- Адаптивные изображения с использованием `<picture>` и `srcset`

#### Доступность (a11y) ✅
- Все изображения имеют атрибут `alt` с описательным текстом
- Все формы имеют связанные `<label>` элементы
- Видимая подсветка фокуса для всех интерактивных элементов (`:focus`, `:focus-visible`)
- ARIA атрибуты: `aria-label`, `aria-required`, `aria-describedby`, `aria-live`
- Контрастность цветов ≥ 4.5:1 (проверено для основного текста)
- Полная навигация с клавиатуры (Tab, Enter, Space)
- Skip link для быстрого перехода к основному содержимому

#### Качество кода ✅
- CSS Custom Properties (переменные) для цветов, отступов, типографики
- Организованная структура CSS с комментариями
- Валидный HTML5 и CSS3 код
- Поддержка тёмной темы через `prefers-color-scheme`

### 3. Архитектура вёрстки

#### Зоны Flexbox:
- **Навигация (nav)**: Flexbox с `flex-direction: column` для мобильных, переключается на `row` для планшетов и десктопов
- **Футер (footer)**: Flexbox для выравнивания элементов

#### Зоны CSS Grid:
- **Портфолио (`.portfolio-grid`)**: 
  - Мобильные: `grid-template-columns: 1fr` (1 колонка)
  - Планшеты: `grid-template-columns: repeat(2, 1fr)` (2 колонки)
  - Десктопы: `grid-template-columns: repeat(3, 1fr)` (3 колонки)
- **Услуги (`.services-grid`)**: 
  - Мобильные: 1 колонка
  - Планшеты и десктопы: 2 колонки
- **Отзывы (`.reviews-grid`)**: 
  - Мобильные: 1 колонка
  - Планшеты: 2 колонки
  - Десктопы: 3 колонки
- **Контакты (`.contacts-container`)**: 
  - Мобильные: 1 колонка
  - Планшеты и десктопы: 2 колонки (информация и форма)

#### Медиазапросы:
```css
/* Базовые стили для мобильных (≤600px) */
/* Стили по умолчанию */

/* Планшеты (601px - 1024px) */
@media (min-width: 601px) { ... }

/* Десктопы (>1024px) */
@media (min-width: 1024px) { ... }

/* Тёмная тема */
@media (prefers-color-scheme: dark) { ... }
```

### 4. Скриншоты работы сайта

> **Примечание:** Для демонстрации работы сайта необходимо сделать скриншоты в браузере на разных размерах экрана. Рекомендуется использовать DevTools браузера (Device Mode) для проверки брейкпоинтов.

- **Мобильная версия (≤600px)**: Одноколоночный layout, вертикальная навигация

- **Десктопная версия (>1024px)**: Трёхколоночный layout для портфолио, максимальная ширина контента 1200px
