# Каталог npm-пакетов

Простое и надёжное веб-приложение для поиска пакетов в официальном реестре **npm**[](https://registry.npmjs.org).

Приложение демонстрирует современные практики фронтенд-разработки:
- поиск в реальном времени с debounce
- пагинация
- кэширование результатов (in-memory + localStorage)
- устойчивость к сетевым ошибкам (retry с backoff, timeout, abort)
- индикаторы загрузки и скелетоны
- удобный UI с обработкой состояний (загрузка, ошибка, пустой результат)

## Возможности

- **Мгновенный поиск** по названию, описанию и ключевым словам пакетов npm
- **Пагинация** результатов (по 10 пакетов на странице)
- **Кэширование** результатов на 1 минуту (быстрый повторный поиск без запроса к серверу)
- **Принудительное обновление** (кнопка «Обновить») — игнорирует кэш и делает свежий запрос
- **Устойчивость к сбоям**:
    - Автоматические повторные попытки при сетевых ошибках
    - Таймаут запросов
    - Отмена предыдущих запросов при новом вводе
- **Скелетон-загрузка** и индикаторы состояния
- **Полностью на vanilla JavaScript** — без фреймворков и зависимостей
