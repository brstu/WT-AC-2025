# Приложение «Коллекция цитат»

Простое, доступное и адаптивное веб-приложение для просмотра, поиска и управления коллекцией известных цитат.  
Поддерживает фильтрацию по категориям, поиск, лайки, удаление цитат и удобное поделение ими.

## Особенности

- **Фильтрация по категориям** через доступные табы (Все / Жизнь / Любовь / Работа)
- **Поиск** по тексту цитаты и автору в реальном времени
- **Лайки** — отметка понравившихся цитат с визуальной обратной связью и tooltip-подсказками
- **Удаление** цитат из коллекции
- **Поделиться** цитатой:
  - Копирование в буфер обмена
  - Публикация в Twitter (X)
  - Отправка по электронной почте
- **Доступность (a11y)**:
  - Полноценная поддержка клавиатуры
  - ARIA-атрибуты для табов, аккордеона, кнопок и модального окна
  - Ловушка фокуса (focus trap) в модальном окне
  - Управление фокусом при открытии/закрытии модалки
- **Адаптивный дизайн** с бургер-меню для мобильных устройств
- **Валидация формы обратной связи** с живой проверкой и отключением кнопки отправки при ошибках
- **Аккордеон** для дополнительной информации (например, «О проекте»)

## Технологии

- Чистый HTML, CSS и vanilla JavaScript (ES Modules)
- Никаких внешних зависимостей и фреймворков
- Данные хранятся в памяти (в массиве `quotes`)
