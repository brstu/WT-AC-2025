# Лабораторная работа — Вариант 27: «Список городов/мест для путешествий с локальным кэшем»

## 1) Описание
В рамках лабораторной работы реализован одностраничный интерфейс **«Travel places»** (список городов/мест для путешествий) на **чистом JavaScript (ES6+)**, без библиотек и фреймворков.  
Приложение загружает данные из **локального mock API** (`places.json`) и поддерживает требования задания:

- список **10–20+** элементов (реализовано **24**)
- состояния UI: **loading / error / empty**
- **поиск/фильтрация**, **пагинация**
- `fetchWithRetry(url, { retries, backoffMs, timeoutMs })` + **AbortController**
- **локальный кэш с TTL**: `in-memory Map` + `localStorage`
- UX: кнопка **«Обновить»** (игнорирует кэш) + индикатор повторной загрузки (spinner)

## 3) Параметры варианта
- Тема: **Список городов/мест для путешествий**
- Источник данных: **локальный mock** `places.json` (24 записи)
- Список: карточки места (город/страна/регион/тип/бюджет/описание/теги)
- Поиск: по `name/country/region/type/budget/tags`
- Фильтры: `Region`, `Type`, `Budget`

  - timeout: **2500ms**
