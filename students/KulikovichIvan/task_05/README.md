# **Вариант 11 — Профили пользователей и портфолио**

API для управления пользовательскими профилями и их проектами в портфолио с поддержкой публичного/приватного доступа, фильтрации по навыкам, локациям и технологиям.

## **Цель работы**

Разработать RESTful API для управления профилями пользователей и их портфолио с реализацией:

- Полного CRUD для профилей и портфолио
- Валидации данных с помощью Zod
- Централизованной обработки ошибок
- Swagger UI документации
- In-memory хранения данных
- Расширенных возможностей фильтрации и поиска

## **1. Архитектура проекта**

Проект построен по модульной архитектуре:
├── server.js # Основной файл приложения
├── package.json # Зависимости и скрипты
├── package-lock.json # Лок блокировка версий

## Быстрый старт

### Установка зависимостей

npm install

### Запуск в режиме разработки

npm run dev

### Обычный запуск

npm start
Сервер запускается на [`http://localhost:3000`](http://localhost:3000)

## Основные эндпоинты

### Профили

GET /api/v1/profiles - список профилей с фильтрацией

POST /api/v1/profiles - создать профиль

GET /api/v1/profiles/:id - получить профиль по ID

PATCH /api/v1/profiles/:id - обновить профиль

DELETE /api/v1/profiles/:id - удалить профиль

## Портфолио

GET /api/v1/portfolio - все портфолио

POST /api/v1/portfolio - создать портфолио

GET /api/v1/profiles/:profileId/portfolio - портфолио профиля

## Особенности

Полный CRUD для профилей и портфолио

Валидация всех входных данных

Фильтрация по навыкам, локации, технологиям

Пагинация и сортировка

Централизованная обработка ошибок

Каскадное удаление данных
