# Инструкция по запуску проекта

## Требования

- Node.js версии 14 или выше
- npm

## Установка

1. Перейдите в папку src:

```bash
cd src
```

1. Установите зависимости:

```bash
npm install
```

1. Создайте файл .env из примера:

```bash
copy .env.example .env
```

1. Запустите сервер:

```bash
npm start
```

Сервер будет доступен по адресу: http://localhost:3000

## Использование

1. Откройте браузер и перейдите на http://localhost:3000
2. Зарегистрируйтесь, введя имя пользователя и пароль
3. Войдите в систему с вашими учетными данными
4. Создавайте, просматривайте и удаляйте записи в дневнике

## API эндпоинты

- POST /api/auth/signup - регистрация
- POST /api/auth/login - вход
- GET /api/entries - получить все записи пользователя
- POST /api/entries - создать запись
- GET /api/entries/:id - получить одну запись
- PUT /api/entries/:id - обновить запись
- DELETE /api/entries/:id - удалить запись
