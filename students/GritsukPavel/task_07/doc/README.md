## Ход выполнения работы

### 1. Структура проекта

```
src/
├── pages/
│   ├── ProjectList.jsx      - список проектов
│   ├── ProjectDetail.jsx    - детальная информация
│   ├── ProjectNew.jsx       - создание проекта
│   ├── ProjectEdit.jsx      - редактирование проекта
│   └── NotFound.jsx         - страница 404
├── main.jsx                 - точка входа приложения
├── index.html               - HTML шаблон
├── vite.config.js           - конфигурация Vite
├── package.json             - зависимости
└── .env                     - переменные окружения
```

### 2. Реализованные элементы

- Маршрутизация на React Router v6
- Страницы: список проектов, детальная информация, создание, редактирование, 404
- CRUD операции через JSONPlaceholder API
- Формы добавления и редактирования с базовой валидацией
- Состояния загрузки и ошибок
- Удаление проектов с подтверждением
- Навигация между страницами

## Вывод

В ходе выполнения лабораторной работы было разработано React-приложение для управления портфолио проектов. Реализована маршрутизация с помощью React Router v6, включая страницы списка, детальной информации, создания, редактирования и 404. Выполнены все CRUD операции через JSONPlaceholder API. Созданы формы с базовой валидацией для добавления и редактирования проектов. Освоены навыки работы с React Router, управления состоянием через useState и useEffect, работы с формами и API запросами.
