# Быстрый старт

## Проверка работоспособности

Проект готов к запуску! Выполните следующие команды:

1. Откройте терминал (cmd или PowerShell)

1. Перейдите в папку src:

```
cd src
```

1. Установите зависимости (уже выполнено):

```
npm install
```

1. База данных уже инициализирована и заполнена тестовыми данными

1. Запустите сервер:

```
node server.js
```

Вы должны увидеть сообщение:

```
Сервер запущен на порту 3000
```

## Тестирование

Откройте второй терминал и проверьте работу API

### Проверка корневого маршрута

```bash
curl http://localhost:3000/
```

### Вход с тестовыми данными

```bash
curl -X POST http://localhost:3000/api/auth/login -H "Content-Type: application/json" -d "{\"email\":\"test@test.com\",\"password\":\"password123\"}"
```

Сохраните полученный token и используйте его для дальнейших запросов:

### Получение заметок

```bash
curl -X GET http://localhost:3000/api/notes -H "Authorization: Bearer ВАШ_ТОКЕН"
```

Проект работает!
