# Фитнес Упражнения

Веб-приложение для просмотра фитнес-упражнений.

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

## Тестирование

```bash
npm test
```

## Docker

```bash
docker-compose up --build
```

## CI/CD

Проект использует GitHub Actions для автоматической проверки кода.
