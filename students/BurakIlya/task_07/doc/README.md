## Цель работы

Создать полноценный SPA на React с маршрутизацией, управлением состоянием, формами и работой с данными. Реализовать CRUD операции для каталога фитнес-упражнений.

## Ход выполнения работы

### 1. Структура проекта

```
task_07/
├── doc/
│   ├── README.md
│   └── screenshots/
│       ├── screenshot1.png
│       ├── screenshot2.png
│       └── screenshot3.png

    ├── pages/
    │   ├── ExerciseList.jsx
    │   ├── ExerciseDetail.jsx
```

**Описание файлов:**

- `main.jsx` — точка входа приложения
- `App.jsx` — главный компонент с роутингом и управлением состоянием
- `pages/ExerciseList.jsx` — страница со списком упражнений
- `pages/ExerciseDetail.jsx` — детальная информация об упражнении
- `pages/ExerciseForm.jsx` — форма создания/редактирования упражнения
- `pages/NotFound.jsx` — страница 404
- `vite.config.js` — конфигурация Vite
- `.env` — переменные окружения

### 2. Реализованные элементы

**Страницы:**
- Главная страница со списком упражнений
- Страница детальной информации об упражнении
- Страница создания нового упражнения
- Страница редактирования упражнения
- Страница 404 для несуществующих маршрутов

**Маршрутизация:**
- React Router v6
- Маршруты: `/`, `/exercise/:id`, `/new`, `/edit/:id`
- Навигация между страницами

**Функциональность:**
- Просмотр списка упражнений
- Поиск упражнений по названию
- Фильтрация по категориям
- Просмотр детальной информации
- Создание новых упражнений
- Редактирование существующих упражнений
- Удаление упражнений с подтверждением
- Уведомления о выполненных действиях

**Формы:**
- Валидация полей (название минимум 3 символа, описание минимум 10 символов)
- Вывод ошибок валидации
- Категории упражнений: Грудь, Ноги, Пресс, Спина, Руки
- Уровни сложности: Начальный, Средний, Продвинутый

**Управление состоянием:**
- Хранение списка упражнений в состоянии App
- Передача данных через props
- Обработка состояний loading/error

## Инструкция по запуску

1. Перейти в папку `src`:

   ```bash
   cd src
   ```

1. Установить зависимости:

   ```bash
   npm install
   ```

1. Запустить проект в режиме разработки:

   ```bash
   npm run dev
   ```

1. Открыть в браузере: `http://localhost:5173`

1. Для сборки production версии:

   ```bash
   npm run build
   ```

## Вывод

В ходе выполнения лабораторной работы было создано SPA приложение на React для каталога фитнес-упражнений. Реализована маршрутизация с использованием React Router v6, включающая страницы списка, детальной информации, создания, редактирования и 404. Внедрены все CRUD операции: создание, чтение, обновление и удаление упражнений. Добавлена базовая валидация форм, поиск и фильтрация упражнений. Использованы технологии: React 18, React Router v6, Vite, React Hook Form. Управление состоянием реализовано через props и локальное состояние компонентов.
