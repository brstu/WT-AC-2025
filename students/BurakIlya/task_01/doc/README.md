## Цель работы

Освоить семантическую разметку HTML5, реализовать адаптивную вёрстку по схеме mobile-first с 2+ брейкпоинтами и повысить качество интерфейса и доступности.

---

### Вариант №29

**Тема:** Локальный гид по экотропам/веломаршрутам: карты, точки, советы по безопасности.

## Ход выполнения работы

### 1. Структура проекта

- `src/index.html` — основная страница сайта
- `src/styles.css` — стилизация и медиазапросы  
- `src/img/` — изображения для маршрутов и точек

### 2. Реализованные элементы

- HTML5 разметка с использованием div-элементов
- Flexbox для организации контента
- Медиазапросы для 3 брейкпоинтов (≤600px, 601-1024px, >1024px)
- Атрибуты `alt` для изображений
- Базовая навигация по якорным ссылкам
